# Sales Prediction Using Python

## Objective
Predict product sales based on advertising expenditures using Machine Learning.

## Dataset
Advertising Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features Used
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

## Model Used
Linear Regression

## Evaluation Metrics
- MAE: 1.46
- R² Score: 0.8994

## Outcome
Successfully built a sales prediction model using advertising data. The model achieved an R² score of 89.94%, indicating strong predictive performance.
