# Movie Rating Prediction

## Objective
Predict movie ratings using machine learning based on movie genre, director, and actors.

## Dataset
IMDb Movies India Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Data Preprocessing
- Removed missing values
- Encoded categorical features using LabelEncoder

## Model Used
Random Forest Regressor

## Evaluation Metrics
- MAE: 1.03
- R² Score: 0.074

## Features Used
- Genre
- Director
- Actor 1
- Actor 2
- Actor 3

## Outcome
Built a movie rating prediction model and evaluated its performance using regression metrics.
