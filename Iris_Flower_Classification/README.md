# Iris Flower Classification

## Objective
Classify Iris flowers into different species using Machine Learning.

## Dataset
Iris Flower Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features Used
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Model Used
Random Forest Classifier

## Accuracy
100%

## Outcome
Successfully classified Iris flowers into their respective species with 100% accuracy.
