# CODSOFT Data Science Internship

This repository contains the projects completed during the CODSOFT Data Science Internship.

## Projects

### ✅ Task 1: Titanic Survival Prediction
Predicting passenger survival using Machine Learning.

### ✅ Task 2: Movie Rating Prediction
Predicting movie ratings using regression techniques.

### ✅ Task 3: Iris Flower Classification
Classifying iris flowers into different species using Machine Learning.

### ✅ Task 4: Sales Prediction Using Python
Predicting sales based on TV, Radio, and Newspaper advertising budgets.

### ✅ Task 5: Credit Card Fraud Detection
Detecting fraudulent credit card transactions using Machine Learning.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook
